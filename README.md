# 🗑️ 垃圾分类大作战 - 微信小游戏

## 项目简介

一款垃圾分类教育小游戏，玩家需要在限定时间内将垃圾拖拽到正确的垃圾桶中。

## 游戏特色

- 🎮 简单易上手的拖拽玩法
- 📚 9个关卡，难度递进
- 🎭 丰富的人物表情反馈
- 🚚 垃圾车进度条倒计时
- 🏆 连击系统，挑战高分
- 📱 震动+音效反馈

## 玩法说明

1. 垃圾出现在屏幕底部区域
2. 从底部拾取垃圾，向上拖拽到对应垃圾桶
3. 正确投放得分，错误投放扣分
4. 在时间内投完所有垃圾且分数≥0即过关

## 垃圾分类

| 桶 | 颜色 | 垃圾类型 |
|---|---|---|
| ♻️ 可回收 | 蓝色 | 塑料瓶、纸盒、玻璃瓶、旧衣物等 |
| 🍎 湿垃圾 | 棕色 | 菜叶、果皮、剩饭、蛋壳等 |
| 🔋 有害 | 红色 | 电池、过期药、灯泡、温度计等 |
| 🗑️ 干垃圾 | 灰色 | 烟蒂、骨头、脏抹布、陶瓷等 |
| 🟢 其他 | 墨绿色 | 其他垃圾 |

## 使用方法

1. 打开 **微信开发者工具**
2. 导入项目目录 `垃圾分类游戏/`
3. 填入你的小游戏 AppID（或使用测试号）
4. 点击编译即可预览
